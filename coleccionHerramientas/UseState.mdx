`useState` es un hook fundamental en React que permite a los componentes funcionales tener y gestionar su propio estado. Antes de los hooks, solo los componentes de clase podían tener estado, pero con `useState`, los componentes funcionales también pueden hacerlo de manera sencilla.

## Importación

Para usar `useState`, primero debes importarlo desde React:

```jsx
import React, { useState } from 'react';
```
